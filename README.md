# Lofi Wear
## WHY?
Z důvodu větší tržby a jednoduššího výběru produktů z domova.
## WHAT?
Zdigitalizování existující kamenné prodejny second hand. Možnost zakoupení produktů online.
## WHO?
* Admin
* Prodejce
* Zákazník
## WHERE?
* Webová stránka
* Mobilní aplikace
## WHEN?
Když lidé budou chtít koupit nebo prodat nějaký produkt.
## HOW?
* Fresh Implementation

# Usecase

![image](https://user-images.githubusercontent.com/53978671/205498305-f59e4be4-4813-4c0f-a8d9-06a21b030f6f.png)

UC2: Parametry
•	Název: Login
•	Kontext: Přihlášení uživatele do systému.
•	Primary actor: User, Reseller, Administrator
•	Stakeholders and Interests: Administrator
•	Precondition: Uživatel musí mít založený účet v databázi. Následně musí zadat správné jméno a heslo.

UC3: Parametry
•	Název: Add products
•	Kontext: Přidání nové položky do databáze.
•	Primary actor: Reseller
•	Stakeholders and Interests: Administrator
•	Precondition: Musí být funkční databáze a uživatel musí být alespoň reseller.

UC4: Parametry
•	Název: Search products
•	Kontext: Vyhledání zadaného produktu v databázi
•	Primary actor: Buyer
•	Stakeholders and Interests: Reseller, Administrator
•	Precondition: Musí být funkční databáze
